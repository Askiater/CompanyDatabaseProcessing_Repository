Полный список SQL запросов для создание полностью идентичной базы данных содержитсья в Full_SQL_Query
Остальные файлы, представяют из себя части этого запроса, а именно:

**Основные
Add_Data_Procedure - создание процедуры для добавления значений в БД
Delete_Data_Procedure - создание процедуры для удаление значений из БД
Find_Data_Procedure - создание процедуры для нахождения значения в БД
Get_All_Data_Procedure - создание процедуры предназначенной для вывода всех значений из БД
Replace_Data_Procedure - создание процедуры для замен фиксированного значения в БД

**Вспомогательные, не содержащиеся в Full_SQL_Query
Query_For_Reupload_Data - создание процедуры для перезагрузки изначальных тестовых данных в БД
